# BotWars Participant Reference Repo

This repo is an example of how you should set up your repo so that your code 
can be pulled by the BotWars team and evaluated. 

Here's how to set up your repo for BotWars in 3 easy steps:

1. Create a repo with the name `botwars2026` on your team leader's github profile
2. Create the folder names as you see them in this repo 
3. Make sure that you move the relevant files to the correct folders and name them with your team name.

For example if your team name is "Team Rainbow Sunshine", your repo structure should look something like this:

```
├── qualifiers
│   └── team_rainbow_sunshine.py
└── README.md

```

Make sure to use the same name for every new game and look up what your file name should be from the excel sheet shared with you.
